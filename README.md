# cozy-voice-chat

Войс-чат для использования в локальной сети.

## Инструкция для хоста

1. Установите зависимости

```
cd ./backend npm i
cd ../frontend npm i
cd ..
```

2. Запустите проекты

```
npm run start:host
```

## Инструкция для клиента

1. Перейдите по ссылке, которую сгенерирует фронт хоста (IP хоста + порт)
2. Вы великолепны
