# cozy-voice-chat

Войс-чат для использования в локальной сети.
В данной версии используется подключение по коду комнаты.

## Инструкция для хоста

1. Установите зависимости

```
cd ./backend npm i
cd ../frontend npm i
cd ..
```

2. Запустите проекты

Терминал 1:
```
cd .\backend\
npm run start
```

Терминал 2:
```
cd .\frontend\
npm run peerServer:start
```

Терминал 3:
```
cd .\frontend\
npm run dev
```
## Инструкция для клиента
